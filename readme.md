# 문제 해결 방법
***

## 문제 1 : 경력인증시 오류 발생
해결 방법 : 카카오톡 지갑 인증요청을 응답했는지 확인합니다.

## 문제 2 : (회원가입 및 정보 수정)기술 등록시 오류 발생
해결방법 : 기술의 연차를 0으로 설정했는지 확인한다 (본 서비스에서는 1년차부터 계산합니다.)

## 문제 3 : 미팅 신청을 할 수 없는 경우
해결방법 : 미팅 가능시간이 설정되어있는지 확인합니다.
